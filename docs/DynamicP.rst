===================
Dynamic Programming
===================

A place for implementation of greedy algorithms

Features
--------

* To see all the available functions in a module, you can just type ``help()`` with the module name as argument. For example,

.. code:: python

    >>> from algopy import greedy_algorithm
    >>> help(greedy_algorithm)
        Help on package algopy.dynamic_programming in algopy:

        NAME
        algopy.dynamic_programming - Collection for dynamic programming algorithms

        PACKAGE CONTENTS
        binary_knapsack
        lis

        DATA
        __all__ = ['binary_knapsack', 'lis']
