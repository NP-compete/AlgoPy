=======
Strings
=======

A place for implementation of string algorithms

Features
--------

* To see all the available functions in a module, you can just type ``help()`` with the module name as argument. For example,

.. code:: python

    >>> from algopy import strings
    >>> help(strings)
        Help on package algopy.strings in algopy:

        NAME
        algopy.strings - Collection of string methods and functions

        PACKAGE CONTENTS
        anagram
        isogram
        manacher_algorithm
        palindrome
        pangram
