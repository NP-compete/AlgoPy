=================
Greedy Algorithms
=================

A place for implementation of greedy algorithms

Features
--------

* To see all the available functions in a module, you can just type ``help()`` with the module name as argument. For example,

.. code:: python

    >>> from algopy import greedy_algorithm
    >>> help(greedy_algorithm)
        Help on package algopy.greedy_algorithm in algopy:

        NAME
        algopy.greedy_algorithm - Collection for greedy algorithms

        PACKAGE CONTENTS
        activity_selection
        fractional_knapsack

        DATA
        __all__ = ['fractional_knapsack', 'activity_selection']
